# Masjid Alfatiha

React single-page application for a masjid / mosque landing site built with Vite.

## Folder structure

- `src/main.tsx`: React entry with router
- `src/App.tsx`: App routes
- `src/layouts/MainLayout.tsx`: Shared layout (navbar + footer)
- `src/pages/HomePage.tsx`: Landing page for Masjid Alfatiha
- `src/components/Navbar.tsx`, `src/components/Footer.tsx`: UI components
- `src/styles/global.css`: Global styling and layout

### Getting started

```bash
npm install
npm run dev
```

Then open the printed localhost URL in your browser to see **Masjid Alfatiha**.
